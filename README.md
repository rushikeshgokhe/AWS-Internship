# Cloud Internship – AWS Skill Builder (Cloud Practitioner)

This repository documents my **cloud internship learning and hands-on lab experience** completed through **AWS Skill Builder (Cloud Quest – Cloud Practitioner)**.

The internship focused on building **foundational AWS Cloud knowledge with guided hands-on labs**, aligned with entry-level **Cloud and DevOps roles**.

---

## Internship Details
- **Role:** Cloud Intern (Virtual / Training-based)
- **Platform:** AWS Skill Builder – Cloud Quest
- **Duration:** 1 Month
- **Mode:** Hands-on Labs & Guided Projects

---

## Technologies & Services Used

### Core AWS Services
- Amazon EC2
- Amazon S3
- Amazon VPC
- AWS IAM
- AWS CloudWatch
- Elastic Load Balancer
- Auto Scaling
- DynamoDB (NoSQL – Basics)
- Amazon RDS (Multi-AZ – Basics)
- Amazon EFS (Basics)

---

## Hands-on Work Performed
- Launched and configured EC2 instances
- Designed VPCs with public and private subnets
- Implemented VPC peering between multiple VPCs
- Configured IAM users, groups, and least-privilege policies
- Set up Auto Scaling and scheduled scaling policies
- Deployed static websites using Amazon S3
- Monitored resources using Amazon CloudWatch
- Performed basic NoSQL operations using DynamoDB
- Implemented high-availability concepts using Multi-AZ architecture
